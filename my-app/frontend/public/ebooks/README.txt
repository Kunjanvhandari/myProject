LibriVista E-Books Directory

To add ebooks to the download section:

1. Place your ebook files (PDF, EPUB, etc.) in this folder
2. Update the file paths in app\page.jsx in the ebooks array

Current ebook entries:
- javascript-the-good-parts.pdf (for "JavaScript: The Good Parts")
- clean-code.epub (for "Clean Code")
- design-patterns.pdf (for "Design Patterns")
- pragmatic-programmer.epub (for "The Pragmatic Programmer")

Note: These are sample entries. You need to add actual ebook files to enable downloads.
